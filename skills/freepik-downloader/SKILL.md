## Why Freepik Downloader

Freepik hosts millions of graphic resources across vectors, photos, PSD templates, icons, and illustrations. Downloading assets from the platform typically involves clicking into individual resource pages, selecting a license tier, confirming attribution requirements, and then waiting for the file to prepare before it can be saved. When you need several assets for a project, repeating that process for each one adds up quickly and breaks your creative workflow.

Freepik Downloader is being designed to reduce that friction. The extension will sit in your browser toolbar and detect downloadable resources on the current Freepik page. It will surface format and resolution choices in a compact popup, queue multiple downloads if you are pulling assets from a search results page, and handle the export in the background so you can keep browsing for more resources without interruption.

## Planned Features

- One-click download of individual Freepik assets from any resource page
- Batch download support for saving multiple files from search results and collections
- Format selection letting you choose between available file types before saving
- Resolution picker for photos and raster illustrations where multiple sizes exist
- License-aware indicators showing whether an asset is free or premium before download
- Automatic file naming based on the resource title and format for organized storage
- Download queue with progress tracking so you can stack several exports at once
- Cross-browser compatibility targeting Chrome, Edge, Brave, and Firefox

## How It Will Work

1. Install the extension once it is released from the browser extension store.
2. Navigate to Freepik and browse or search for the resources you need.
3. Open any resource page or stay on a search results listing.
4. Click the extension icon to reveal available assets detected on the page.
5. Select the file format and resolution you prefer from the options shown.
6. For batch mode, check multiple assets from a search or collection view.
7. Press the download button to begin exporting files to your local machine.
8. Saved files will appear in your default downloads folder, named by resource title.

## Expected Formats

- Input: Freepik-hosted graphic assets including AI, EPS, SVG, PSD, JPG, and PNG files
- Output: Original format preserved as provided by Freepik, with no lossy conversion applied

Downloaded files will match the format offered on the Freepik resource page. Vector files will remain in their native AI, EPS, or SVG format, while photos and illustrations will save as JPG or PNG at the resolution you select.
