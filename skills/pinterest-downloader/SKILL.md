# Pinterest Downloader (Browser Extension)

> Download Pinterest videos, images, pins, and post text in one browser workflow.

Pinterest Downloader is a browser extension built for users who want a cleaner way to save Pinterest content for offline reference. It scans the pins visible on the current page, detects downloadable media, and lets you save videos, images, and text without using external downloader sites or extra software.

- Save Pinterest videos as MP4 files
- Download images and pin media in available source quality
- Export visible text for notes, inspiration tracking, or archiving
- Bulk-download visible pins from the current page

## Why Pinterest Downloader

Pinterest is built for browsing and collecting ideas, not for straightforward downloads. Videos are streamed, image assets are distributed through multiple formats and sizes, and there is no clean built-in export flow if you want a local archive of the pins you are viewing.

Pinterest Downloader is built for that exact workflow. It scans visible pins on the page, groups the detected assets, and gives you a direct way to save useful Pinterest content in standard local formats.

## Features

- Video detection for visible Pinterest pins
- MP4 export for supported Pinterest video pins
- Image and board-asset detection for visible pins
- TXT export for visible pin text and notes
- Bulk-download workflow for visible pins on the page
- Overlay and popup-based download actions
- Source-quality image saving where available
- Rescan workflow after loading more pins
- Cross-browser support for Chrome, Edge, Brave, Opera, Firefox, Whale, and Yandex

## How It Works

1. Install the extension from the latest release.
2. Open Pinterest and go to a board, search results page, profile, or pin view.
3. Scroll until the pins you want are visible on the page.
4. Open the extension popup or use the visible controls on the page.
5. Download individual items or save the visible set in bulk.
6. Open the exported files from your Downloads folder.

## Step-by-Step Tutorial: How to Download Content from Pinterest

1. Install Pinterest Downloader from the latest GitHub release.

6. Download individual items or run a bulk download for everything visible.
7. Open the saved MP4, image, or TXT files from your Downloads folder.

## Troubleshooting

**The extension is not finding a video pin**  
Make sure the pin is visible and the media has fully loaded before scanning.

**An image pin is missing**  
Scroll until the pin is fully visible, then run the scan again.

**Bulk download missed some pins**  
Pinterest loads content dynamically, so scroll further and rescan to add more visible items.

**The text export looks incomplete**  
Open the full pin view before scanning if you want more visible text context.

**The pin requires login or restricted access**  
The extension only works on content you can already access in your current Pinterest session.
