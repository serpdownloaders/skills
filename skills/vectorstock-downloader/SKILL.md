## Why VectorStock Downloader

VectorStock hosts millions of royalty-free vector images, but downloading purchased assets can involve multiple clicks through confirmation screens, license pages, and individual file links. When you are working on a project that requires dozens of vectors, the repetitive navigation adds up quickly. If you purchased assets across different sessions, tracking down every file in your download history becomes its own chore.

VectorStock Downloader is being designed to sit inside the browser and connect directly with VectorStock's website. The goal is to detect your licensed assets on supported pages you visit, let you select the files you need, and pull them to your local machine in a single streamlined action rather than a per-file download loop.

## Planned Features

- One-click download of licensed vector files from any VectorStock page
- Batch asset retrieval for pulling multiple vectors from a collection or search
- Automatic detection of purchased and licensed items on the current page
- Folder organization options to sort downloads by project, category, or date
- Format selection when multiple file types are available for a single asset
- Download history log so you can track which assets have already been saved
- Browser-native operation with no external software or plugins required
- Cross-browser compatibility targeting Chrome, Edge, Brave, and Firefox

## How It Will Work

1. Install the extension once it is released.
2. Browse VectorStock and sign in to your account with active licenses.
3. Navigate to any vector image, collection page, or your purchase history.
4. The extension will detect available licensed assets on the current page.
5. Open the extension popup to view the list of downloadable files.
6. Select individual assets or choose all items for a batch download.
7. Pick your preferred file format if the asset offers multiple options.
8. Confirm the download and the files will be saved to your chosen local folder.

## Supported File Types

- Input: VectorStock licensed assets (EPS, SVG, AI, JPEG previews)
- Output: Original vector formats preserved as-is during download

Downloaded files will retain their original format and resolution. Vector formats like EPS and SVG remain fully editable in standard design applications such as Adobe Illustrator, Inkscape, Affinity Designer, and CorelDRAW.
